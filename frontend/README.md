# MCTJK Frontend (Next.js)

Высокопроизводительный интерфейс платформы MCTJK с упором на визуальное совершенство и кинематографичный опыт.

## 🚀 Стек технологий
- **Next.js 15+**: React фреймворк с поддержкой App Router.
- **GSAP & ScrollTrigger**: Сложные анимации и контроль скролла.
- **Lenis**: Плавная прокрутка (Smooth Scroll).
- **Tailwind CSS**: Утилитарный CSS фреймворк.
- **Framer Motion**: Микро-взаимодействия.
- **Lucide React**: Коллекция иконок.

## 🛠 Установка

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите в режиме разработки:
   ```bash
   npm run dev
   ```

## 🎨 Дизайн-система
Проект использует философию **Tech-Luxury**:
- Тёмная тема с золотыми акцентами.
- Глубокое размытие (Backdrop Blur).
- Кастомные шрифты (Inter, Outfit).
- Плавные переходы между страницами.

## 📦 Скрипты
- `npm run dev`: Запуск локального сервера.
- `npm run build`: Сборка проекта для production.
- `npm run start`: Запуск собранного приложения.
- `npm run lint`: Проверка кода линтером.
